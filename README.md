# How to use this?

```$ sudo apt update -y && sudo apt upgrade -y && sudo apt dist-upgrade -y && flatpak update -y ```

```$ git clone https://github.com/bossman7309/NovaThemeInstaller.git```

```$ cd NovaThemeInstaller```

```$ sudo ./Themes.sh```

```$ sudo ./Icons.sh```
